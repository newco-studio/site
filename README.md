# Newco Studio — Site

Static marketing site for Newco Studio, published via GitHub Pages.

**Live site:** https://newco-studio.github.io/site/

Plain HTML/CSS/JS — no build step. Bilingual (FR/EN) with a language toggle.

## Structure

- `index.html` — Home
- `use-cases.html` — Use cases
- `platform.html` — Platform
- `services.html` — Services (AI+ Studio)
- `contact.html` — Contact
- `styles.css`, `app.js`, `images/` — assets
